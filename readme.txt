Dit programma is geschreven om primers uit een sequentie te halen. Wanneer bij het 
invoeren de correcte informatie word opgegeven en er zich primers bevinden in de
sequentie word er naast de gevonden primers ook het GC%, TM en
lengte van het PCR product weergeven per gevonden primerpaar.

Dit programma is geschreven in python 3.6. Er word aangeraden om pycharm te installeren
en te gebruiken voor dit script. Om dit programma te gebruiken moeten er een aantal 
modules worden ge�nstalleerd. 
Deze modules zijn:
           - wx
	- easygui

Om dit te installeren moet er in de python terminal (deze kan links onder gevonden
worden tussen "TODO" en "Python Console") de volgende commands worden uitgevoerd:

pip install wx;
pip install easygui

Als deze modules zijn ge�nstalleerd werkt het script correct.

Om het programma te starten moet het script BMOLTEC_overkoepeling.py uitgevoerd worden.

Voor verdere hulp over de inhoud kan er op de "Help" knop gedrukt worden in het 
programma.
